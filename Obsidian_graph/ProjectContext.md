# ProjectContext (Глобальное состояние)

Краткое описание: Провайдер состояния (Context API), который абстрагирует работу с базой данных Firebase и предоставляет данные всему приложению.

## ⚙️ Зависимости (Что использует)
- [[firebase]] — инициализация SDK, Firestore (коллекции `projects`, `baths`, `portfolio`)
- [[project]] — TypeScript интерфейсы (типы) для типизации данных

## ⬅️ Кем используется (Обратные связи)
- [[App]] — оборачивает всё приложение в `ProjectProvider`
- [[Home]] — получение популярных проектов
- [[Catalog]] — получение всех проектов домов
- [[CatalogBaths]] — получение всех проектов бань
- [[ProjectDetails]] — получение данных конкретного дома
- [[BathDetails]] — получение данных конкретной бани
- [[Portfolio]] — получение списка выполненных работ
- [[Admin]] — получение данных и вызов функций для их изменения (CRUD)
