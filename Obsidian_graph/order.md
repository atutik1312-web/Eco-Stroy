# Order Types (order.ts)

Краткое описание: TypeScript интерфейсы, описывающие структуру заявок (лидов), поступающих с сайта, а также их возможные статусы для встроенной CRM-системы.

## 🧱 Основные сущности
- `Order` (Заявка/Лид: содержит контакты, дату, источник и детали расчета)
- `OrderStatus` (Статусы: `new`, `in_progress`, `done`, `rejected`)

## ⬅️ Кем используется (Обратные связи)
- [[Admin]] — типизация списка заявок во вкладке CRM и строгий контроль изменения статусов
