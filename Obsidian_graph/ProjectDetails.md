# ProjectDetails (Детальная страница дома)

Краткое описание: Страница с подробной информацией о выбранном проекте дома (галерея, планировки, комплектации, цены).

## 🧱 Основные секции / Компоненты
- Галерея изображений (с использованием lightbox)
- Блок характеристик (площадь, этажи, спальни)
- Таблица комплектаций (Базовая, Теплый контур, Под ключ)
- Планировки этажей
- Форма заявки на консультацию/покупку

## 🔗 Навигация (Куда можно перейти)
- [[Catalog]] — кнопка "Назад в каталог"

## ⚙️ Зависимости (Логика и данные)
- [[ProjectContext]] — получение данных конкретного проекта по ID
- [[telegram]] — отправка заявки со страницы проекта
- [[email]] — дублирование заявки на email
