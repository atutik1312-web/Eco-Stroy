# Login (Страница входа)

Краткое описание: Страница аутентификации для доступа к панели управления (Admin).

## 🧱 Основные секции / Компоненты
- Форма входа
- Кнопка "Войти через Google"
- Блок вывода ошибок авторизации

## 🔗 Навигация (Куда можно перейти)

## ⚙️ Зависимости (Логика и данные)
