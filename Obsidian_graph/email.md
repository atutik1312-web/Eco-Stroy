# Email Service (email.ts)

Краткое описание: Сервис для отправки email-уведомлений менеджерам при поступлении новых заявок с сайта (использует EmailJS).

## ⚙️ Зависимости (Что использует)
- Внешнее API EmailJS
- Переменные окружения (VITE_EMAILJS_...)

## ⬅️ Кем используется (Обратные связи)
- [[Home]] — при отправке формы захвата на главной
- [[Contacts]] — при отправке формы обратной связи
- [[Calculator]] — при отправке результатов расчета
