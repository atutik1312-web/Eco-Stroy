# Firebase (firebase.ts)

Краткое описание: Конфигурационный файл для инициализации Firebase SDK (Firestore для базы данных и Auth для аутентификации).

## ⚙️ Зависимости (Что использует)
- Внешнее API Firebase
- Переменные окружения (VITE_FIREBASE_...)

## ⬅️ Кем используется (Обратные связи)
- [[ProjectContext]] — для работы с коллекциями каталога
- [[Admin]] — для работы с коллекцией заявок (`orders`) и выхода из аккаунта
- [[Login]] — для входа через Google
- [[ProtectedRoute]] — для проверки состояния авторизации пользователя
