# ProtectedRoute (ProtectedRoute.tsx)

Краткое описание: Компонент высшего порядка (HOC) для защиты приватных маршрутов. Проверяет, авторизован ли пользователь.

## ⚙️ Зависимости (Что использует)
- [[firebase]] — подписка на изменение состояния авторизации (`onAuthStateChanged`)

## ⬅️ Кем используется (Обратные связи)
- [[App]] — оборачивает маршрут `/admin`
- [[Admin]] — защищаемый компонент (дочерний)
- [[Login]] — куда перенаправляется неавторизованный пользователь
