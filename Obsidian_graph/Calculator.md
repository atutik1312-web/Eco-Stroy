# Calculator (Калькулятор стоимости)

Краткое описание: Интерактивный многошаговый калькулятор (wizard) для примерного расчета стоимости строительства.

## 🧱 Основные секции / Компоненты
- Шаги выбора параметров (площадь, этажность, тип крыши, фундамент, отделка)
- Динамический блок "Итоговая стоимость" (пересчитывается на лету)
- Финальная форма сбора контактов для отправки сметы
