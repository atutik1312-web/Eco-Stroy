# Project Types (project.ts)

Краткое описание: TypeScript интерфейсы и типы данных, описывающие структуру проектов домов, бань и портфолио для обеспечения строгой типизации во всем приложении.

## 🧱 Основные сущности
- `Project` (Каркасный дом)
- `BathProject` (Деревянная баня)
- `PortfolioItem` (Выполненная работа)
- `ProjectConfig` (Комплектации и цены)

## ⬅️ Кем используется (Обратные связи)
- [[ProjectContext]] — типизация глобального состояния и функций CRUD
- [[Catalog]] — типизация карточек в списке домов
- [[CatalogBaths]] — типизация карточек в списке бань
- [[ProjectDetails]] — типизация данных конкретного дома
- [[BathDetails]] — типизация данных конкретной бани
- [[Portfolio]] — типизация списка выполненных работ
- [[Admin]] — типизация форм редактирования и создания новых объектов
