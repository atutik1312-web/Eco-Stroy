# Portfolio (Портфолио)

Краткое описание: Галерея успешно реализованных проектов (построенных объектов) для демонстрации опыта компании.

## 🧱 Основные секции / Компоненты
- Сетка выполненных работ
- Карточки объектов (фото, описание, сроки, площадь)
- Lightbox для просмотра фото в полном размере

## 🔗 Навигация (Куда можно перейти)
- [[Home]] — возврат на главную
- [[Contacts]] — призыв к действию (связаться с нами)

## ⚙️ Зависимости (Логика и данные)
- [[ProjectContext]] — получение списка выполненных работ
