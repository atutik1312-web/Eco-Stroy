# Admin (Панель управления)

Краткое описание: Защищенная CMS/CRM панель для управления каталогом (дома, бани, портфолио) и обработки заявок (лидов).

## 🧱 Основные секции / Компоненты
- Вкладка "Проекты домов" (список и редактор)
- Вкладка "Бани" (список и редактор)
- Вкладка "Портфолио" (список и редактор)
- Вкладка "Заявки" (CRM-доска с лидами и статусами)
- Кнопки Экспорта/Импорта базы данных (Backup)

## 🔗 Навигация (Куда можно перейти)
- [[Login]] — при нажатии кнопки "Выйти" (Logout)
- [[Home]] — возврат на сайт

## ⚙️ Зависимости (Логика и данные)
- [[ProjectContext]] — вызов функций CRUD (addProject, updateProject, deleteProject)
- [[firebase]] — прямые запросы к коллекции `orders` (onSnapshot, updateDoc, deleteDoc), а также `signOut` для выхода
- [[ProtectedRoute]] — компонент, защищающий доступ к этой странице
