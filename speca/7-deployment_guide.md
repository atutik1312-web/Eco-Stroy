# Specification: Deployment Guide (Firebase Hosting)

This document provides a step-by-step guide to deploying the Eco-Stroy React (Vite) application to Firebase Hosting. Since the application uses React Router for client-side navigation (Single Page Application), specific routing configurations are required.

## Prerequisites

1. **Node.js and npm** must be installed on your local machine.
2. You must have a **Firebase account** and an existing Firebase project (e.g., `ecostroy-app`).

---

## Step 1: Install Firebase CLI

First, you need to install the Firebase Command Line Interface globally on your computer. Open your terminal and run:

```bash
npm install -g firebase-tools
```

## Step 2: Authenticate with Firebase

Log in to your Google account connected to Firebase:

```bash
firebase login
```
*This will open a browser window asking you to grant permissions to the Firebase CLI.*

## Step 3: Build the Project

Before deploying, you need to create an optimized production build of your React application. Run the following command in the root directory of your project:

```bash
npm run build
```
*This command uses Vite to bundle your app and outputs the static files into the `dist` directory.*

## Step 4: Initialize Firebase Hosting

Initialize Firebase in your project directory:

```bash
firebase init hosting
```

You will be prompted with a series of questions. Answer them exactly as follows:

1. **Please select an option:** 
   Select `Use an existing project` and choose your project (e.g., `ecostroy-app`).
2. **What do you want to use as your public directory?** 
   Type `dist` (This is crucial because Vite outputs the build to the `dist` folder, not `public`).
3. **Configure as a single-page app (rewrite all urls to /index.html)?** 
   Type `y` (Yes). *This ensures that React Router handles the routing correctly without returning 404 errors on page reloads.*
4. **Set up automatic builds and deploys with GitHub?** 
   Type `N` (No, unless you want to set up CI/CD right now).
5. **File dist/index.html already exists. Overwrite?** 
   Type `N` (No). *You want to keep the file generated by your `npm run build` command.*

## Step 5: Verify `firebase.json` Configuration

After initialization, a `firebase.json` file will be created in your project root. Ensure it looks exactly like this to guarantee proper SPA routing:

```json
{
  "hosting": {
    "public": "dist",
    "ignore": [
      "firebase.json",
      "**/.*",
      "**/node_modules/**"
    ],
    "rewrites": [
      {
        "source": "**",
        "destination": "/index.html"
      }
    ]
  }
}
```

## Step 6: Deploy to Firebase

Finally, deploy your built application to Firebase Hosting:

```bash
firebase deploy --only hosting
```

Once the deployment is complete, the CLI will output a **Hosting URL** (e.g., `https://ecostroy-app.web.app`). You can click this link to view your live website!

---

## Updating the Site in the Future

Whenever you make changes to your code and want to update the live website, simply run these two commands:

```bash
npm run build
firebase deploy --only hosting
```
